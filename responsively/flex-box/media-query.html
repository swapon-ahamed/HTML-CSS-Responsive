Media Query: Basic Syntacs:

@media () {...}

@media media-type () {...}

@media media-type and (media-features) {...}

The media type let's us target different types of media.

  • Scren @medai screen {...}
  • Print @medai print {...}
  • Speech @medai speech {...}

The media condition let's us target specific conditions within that media type.

  • Widths @media (min-width:600px) { ... }
  • Orientation @media (orientation: landscape) { ... }
  • Specefic features @media (hover) { ... }
Both media types and conditions are optional
We d need to either have a type or condition though.

Example: Target only screen

@media screen { ... }

Width of viewport

@media (min-width:960px) { ... }

Combine a type with condition by using and

@media screen and (min-width: 960px) { ... }

@media screen and (orientation:portrait) { ... }
