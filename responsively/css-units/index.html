CSS Units

We must make a design and have it work on every device, from a watch screen to a 80" TV. We will work our way up from getting a design to simply work on a single screen size to being able to get it to work on any device.

What we will be covering in this module:

  • How to approach a layout
  • CSS units (absolute, relative and percentages)
  • Flex box basics ( our layout tool)
  • Media query basics (adapting to different screen sizes)

There are different types of units that we can use

CSS units come in a few different flavors:

  • Absolute units
  • Relative
  • percentages

Absolute units are the easiest to understand

  • Pixels(px)
  • pt, cm, mm, in etc

Percentage

Percentages are mainly used for width, and are pretty easy to understand

  • Relative to their parent (except on height, where things get weird).

Relative units

There are two types of relative units

  • Relative fo font-size:
    - em and rem (and many other less common ones)
  • Relative fo viewport. Viewport is browser window.
    - vw
    - vh
    - vmin
    - vmax
